# MonetizeMaxxing — Design Guidelines

## Design ethos
Advisor-grade, analyst-memo aesthetic. Calm, confident, credible — not hypey, no startup-marketing gloss. The interface should feel like a premium financial/strategy tool that a VC would be comfortable showing inside an investment committee.

## Color
- Base: near-white / soft off-white background (#FAFAF8) with deep ink text (#11141A).
- Surfaces: clean white cards with subtle borders (#E7E7E2) rather than heavy shadows.
- Primary accent: a deep, trustworthy green (#0F5132 / emerald family) used sparingly for primary actions, ranking emphasis, and key economic figures.
- Secondary/neutral grays for supporting metadata and labels.
- Stage badges: distinct but muted tones (pre-revenue, early traction, scaling) so stage is always legible at a glance.

## Typography
- Headings: a confident serif or strong grotesque sans for an editorial, memo-like authority.
- Body: a highly readable sans (system/Inter-like) for dense reasoning text.
- Use clear hierarchy: rank number, model type as headline, then rationale / economics / experiment as labeled sections.

## Layout & components
- Input: a focused, single-column "company profile" form — description, sector, stage selector (segmented control), current business model. Generous whitespace.
- Output: a ranked list of strategy cards (1, 2, 3...). Each card clearly labels: Model type, Why it fits (this company, this stage), Revenue logic / economics, Validate it (one concrete experiment).
- Ranking should be visually obvious (numeric rank, ordering, subtle emphasis on the top play).
- Presentable mode: output should look clean enough to screenshot or present.

## Elevation & motion
- Minimal elevation; rely on borders and spacing. Subtle, restrained transitions only.
- No flashy gradients or playful illustrations — keep it composed and professional.
